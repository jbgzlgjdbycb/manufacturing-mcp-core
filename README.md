# Manufacturing MCP Core

## 🎯 Vision
Transform "dark capacity" in manufacturing into programmable digital services and build a global network for AI Agent autonomous transactions.

## 🚀 Getting Started

```bash
# Clone repository
git clone https://github.com/jbgzlgjdbycb/manufacturing-mcp-core.git
cd manufacturing-mcp-core

# Install dependencies
pip install -e .
```
